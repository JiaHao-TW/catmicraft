# 變更日誌

## [1.0.0] - 2025-05-24

### Added

- 初始項目設置
- Vue 3 前端應用
  - 首頁組件
  - 項目列表頁面
  - Vue Router 集成
  - Pinia 狀態管理
- Go 後端 API
  - 項目 CRUD 操作
  - MongoDB 集成
  - Chi 路由支持
- GitHub Pages 前端部署
- Go 後端部署配置
- 完整的文檔和指南

### Features

- 📌 創建、讀取、更新、刪除產品
- 🎨 響應式設計
- 🔗 RESTful API
- 📦 完整的堆棧項目結構
- 🚀 自動化部署流程
